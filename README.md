## Formats

- ttf
- eot
- woff
- woff2

## Webpack installation

To begin with

```sh
$ npm install typeface-mplus-1p
```

then, import into your sourcecode

```js
require("typeface-mplus-1p");

// OR

import "typeface-mplus-1p";
```
